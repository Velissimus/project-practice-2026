# 4. Выполнение вариативной части практики

## Название проекта

Python: Create a Simple Python Text Editor

---

## Цель работы

Разработка несложного приложения для работы с текстовыми файлами с использованием языка Python.

---

### Используемые технологии

При разработке текстового редактора использовались следующие технологии и инструменты:

- **Python** — язык программирования, использованный для реализации логики приложения.
- **Tkinter** — стандартная библиотека Python для создания графического интерфейса пользователя
- **CustomTkinter** — расширение для Tkinter, использованное для создания современного интерфейса, настройки тем оформления и стилизации элементов.
- **Pillow (PIL)** — библиотека для загрузки, обработки и отображения изображений, применённая для работы с фоновыми изображениями и графическими элементами интерфейса.
- **FileDialog** — инструмент для открытия и сохранения файлов через системные окна выбора.
- **ColorChooser** — компонент для выбора цвета текста пользователем.
- **PyCharm** — среда разработки, использованная для написания и отладки кода.
- 

Программа реализует графический интерфейс текстового редактора с поддержкой изменения тем оформления, форматирования текста, открытия и сохранения файлов, а также настройки визуального отображения элементов.
---

## Основные функции приложения

Разработанный текстовый редактор предоставляет пользователям следующие возможности:

- создание новых текстовых файлов;
- открытие существующих файлов в формате txt;
- сохранение файлов под новым именем;
- изменение оформления текста (жирный, курсив, подчёркивание);
- изменение цвета текста;
- выделение текста цветом;
- переключение между светлой и тёмной темами;
- поддержка горячих клавиш.
