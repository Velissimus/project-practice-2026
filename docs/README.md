# Отчёт по проектной (учебной) практике

---

# Введение

В ходе проектной (учебной) практики выполнялись задачи по освоению инструментов управления версиями, созданию документации, разработке статического сайта проекта и выполнению вариативной части задания.

Практика проводилась в рамках проекта **Dark Machine** — пиксельной игры в стимпанк-сеттинге, действие которой происходит в мире, где механизированные системы заменили человеческий труд.

---

# 1. Описание проекта Dark Machine

## Цель проекта

Разработать альфа-версию игры "Dark Machine", демонстрирующую:

- сложно структурированную карту мира
- механику взаимоисключающих путей развития
- влияние решений игрока на прохождение
- базовую боевую систему

---

## Проблема проекта

Отсутствие в жанре метроидвания систем прогрессии, в которых выбор игрока существенно влияет на доступные способности и порядок исследования игрового мира.

---

## Планируемый результат

Создание альфа-версии игры, включающей:

- базовые игровые механики
- систему выбора способностей
- две игровые локации
- core-геймплей
- базовую боевую систему
